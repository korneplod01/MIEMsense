# MIEMsense - exterhal WH + SDK-набор для CS 1.6

### ВАЖНО: во избежание блокировки VAC запускайте чит **ТОЛЬКО** с параметром -insecure
### Чит стабильно работает только в оконном режиме с включенной верт. синхронизацией; масштаб интерфейса Windows должен быть равен 100%

## Функции

**Основные возможности:**
*   **Актуальные игровые структуры:** структуры игрока (`PlayerInfo`) и объекта камеры (`CS_View`)
*   **Сканирование памяти (SDK):** автоматический поиск дескриптора процесса игры (`hl.exe`), определение базовых адресов игровых модулей (`hw.dll`, `client.dll`)
*   **WorldToScreen:** функция проекции 3D-координат игрового мира на 2D-координаты экрана монитора для работы с трёхмерными векторами и объектами камеры GoldSRC (`CS_View`)
*   **ESP:** функция отрисовки ESP-бокса вокруг модели игрока с настройкой размера и цвета
*   **TeamFromModel:** корректное определение модельки игрока и определение его команды

## Способы установки и сборки

Проект собирается с помощью **CMake** под 32-битную архитектуру (x86).

### Системные требования
* Компилятор: MinGW (32-bit /i686) или MSVC (архитектура x86)
* CMake 3.16+

### Подготовка окружения (Pre-requisites)
Перед началом сборки откройте терминал **MSYS2** и установите 32-битный компилятор, а также адаптированную версию CMake:
```
pacman -S --noconfirm mingw-w64-i686-toolchain mingw-w64-i686-cmake
```

### Пошаговая сборка
Откройте терминал **MSYS2 MinGW 32-bit** и выполните следующие команды:

```
git clone https://github.com/korneplod01/MIEMsense

cd MIEMsense

mkdir build && cd build

cmake -G "MinGW Makefiles" ..

cmake --build .
```

После успешной компиляции вы можете запустить тестирование всех ключевых модулей:

```
ctest -V
```
